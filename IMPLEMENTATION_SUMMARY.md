## 🚀 Как это теперь работает

### Для нового пользователя

```bash
# Шаг 1: Установка
pnpm install

# Шаг 2: Создание конфига
cp .env.development.local.example .env.development.local
# Редактирует файл в любом редакторе

# Шаг 3: Инициализация (вся магия здесь!)
pnpm setup

# Шаг 4: Запуск
pnpm dev
```

**Результат:** Проект работает, БД инициализирована, вход функционирует ✨

### Что делает `pnpm setup`

```
✅ Проверяет наличие .env.development.local

✅ Выводит статус каждой переменной
```
